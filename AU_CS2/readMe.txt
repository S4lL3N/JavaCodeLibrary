Course Outcomes
By the end of this course, you should be able to:

 1. Design, implement and test programs based on sound object-oriented design

 2. Design, implement and test class inheritance hierarchies based on informal specifications

 3. Implement and test class methods using polymorphism based on informal specifications

 4. Implement and test programs that require and use exception handling
