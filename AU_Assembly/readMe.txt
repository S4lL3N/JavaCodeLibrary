Course Outcomes
By the end of this course, you should be able to:

 1. Understand low-level details of a computer

 2. Understand binary representation and Boolean algebra

 3. Apply basic knowledge of digital logic circuits to computer organization

 4. Understand and learn the roles of the CPU, the memory and the I/O system

 5. Understand Internal CPU and operations

 6. Learn and use a CPU instruction set

 7. Rewrite a high level using the CPU instruction set

 8. Manipulate arrays in assembly programming

 9. Understand and use procedures, interrupts
